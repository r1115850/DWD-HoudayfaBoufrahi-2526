# Recipefinder

Single page recipefinder voor de DWD AI-opgave.

## Starten

Open `index.html` in de browser. De toepassing gebruikt de gratis TheMealDB API en heeft geen installatie nodig.

## Inhoud

- `index.html`: HTML-structuur
- `css/style.css`: styling en responsive layout
- `js/script.js`: API-aanroepen, rendering en favorieten
- `AGENTS.md`: agent instructies
- `documentatie.md`: AI-documentatie en testverslag
