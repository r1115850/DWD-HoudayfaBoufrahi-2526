# Agent instructies - Recipefinder

Werk binnen de grenzen van de cursus Dynamic Web Development.

## Technische afspraken

- Gebruik enkel HTML, CSS en gewone JavaScript.
- Gebruik geen JavaScript-frameworks, Bootstrap of externe componentlibraries.
- Haal data op met `fetch` en `async/await` via TheMealDB API.
- Hou de code leesbaar, opgesplitst en becommentarieer enkel waar dat helpt.
- De toepassing moet responsive zijn en bruikbaar blijven op mobiel en desktop.
- Favorieten moeten blijven bestaan na verversen of sluiten van de browser via `localStorage`.

## Functionele checklist

- Zoeken op trefwoord via TheMealDB.
- Resultaten tonen als receptkaarten met afbeelding, naam, categorie en regio.
- Maximaal 20 recepten per zoekopdracht tonen.
- Detailweergave tonen met afbeelding, categorie, regio, instructies en ingredienten.
- Recepten visueel als favoriet markeren.
- Recepten toevoegen aan en verwijderen uit favorieten.
- Feedback tonen bij lege zoekopdracht, geen resultaten en API-fouten.
